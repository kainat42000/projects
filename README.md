# Event Management System (EMS)

This project is a simple **Event Management System** built using **PHP**, **MySQL**, **HTML**, and **CSS**.  
It allows users to **add, view, update, and delete** event details through an easy-to-use interface.

## Features
- Add new events  
- Update and delete existing events  
- Search events by name  
- User login and registration system  

## Tools Used
- XAMPP (Apache & MySQL)
- PHP
- HTML/CSS
- GitHub for version control
